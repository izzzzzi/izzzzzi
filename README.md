## 👋 About Me

Backend / AI developer with **10+ years** of experience — currently building the infrastructure that powers LLM agents in production.

Based in **Moscow** 🇷🇺, working remotely as a developer at SELFSEC. Open-source contributor with **4,500+ contributions/year**.

### What I'm building

| Focus Area | What I do |
|------------|-----------|
| 🧰 **Agent Tool Ecosystem** | Go CLIs that give LLM agents SSH access, browser control, and multi-source search — designed from the ground up for token efficiency and reliable JSON contracts |
| 🔌 **MCP Servers** | Model Context Protocol servers that bridge AI assistants with code wikis, TON blockchain compilers, and external APIs |
| ⛓️ **TON Blockchain** | Non-custodial P2P escrow systems on TON smart contracts (Tolk), full-stack dApps with Telegram Mini Apps |
| 🤖 **AI / LLM Engineering** | Multi-agent systems with adversarial debate, plugin ecosystems for OpenCode/Claude Code, prompt engineering |
| 🐍 **Backend Systems** | High-throughput ETL pipelines, queue-driven architectures, Telegram bot ecosystems |
